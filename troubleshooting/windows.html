Temporary User

This issue occurs when:

  • The user's directory is removed from a domained computer
  • A virus or program edits the registry profile

Fixes

  1. Log in as an administrator
  2. Open the Registry Editor
    • It is always recommended to backup the registry before making changes
  3. Navigate to HKEY_LOCAL_MACHINE \ SOFTWARE \ Microsoft \ WindowsNT \ CurrentVersion \ ProfileList
  4. Rename the Key to remove the .bak or .back from the end of it, or remove the Key
  5. Reboot the system

Unable to change Computer Name

This issue occurs when:

  • N/A - Unknown

Fixes

  1. Open an elevated command prompt
  2. Enter the following command: wmic computersystem where name="%computername%" call rename name="Computer_Name"
  3. Replace Computer_Name with the name you want to set
  4. Reboot the system
  1. Open an elevated PowerShell
  2. Enter the following command: Rename-Computer -NewName "Computer_Name"
  3. Replace Computer_Name with the name you want to set
  4. Reboot the system

Unable to change Workgroup

This issue occurs when:

  • N/A - Unknown

Fixes

  1. Open an elevated command prompt
  2. Enter the following command: wmic computersystem where name="%computername%" call joindomainorworkgroup name="Workgroup_Name"
  3. Replace Workgroup_Name with the name you want to set
  4. Reboot the system
  1. Open an elevated PowerShell
  2. Enter the following command: Add-Computer -WorkGroupName "Workgroup_Name"
  3. Replace Workgroup_Name with the name you want to set
  4. Reboot the system
